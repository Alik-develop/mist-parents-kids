# Changelog — рушій уроків (каркас) + прогрес у кабінеті

_25 червня 2026 · скелет навчального модуля (контент уроків — окремо, з Оксаною)_

Структуру взято за зразком державної ВШО (предмет → клас → урок = відео + матеріал + тест + прогрес), але привʼязано до дитини й видно батькам.

## Що зроблено

- **БД засіяна** прикладами (placeholder-контент, замінимо): 3 уроки (математика ×2, історія) + тести на закріплення. Уроки зроблено **публічними для читання** (контент безкоштовний; прогрес лишається приватним по дитині).
- **`docs/urok.html` — плеєр уроку** (повністю переписано). Завантажує урок за `?id`: відео (або плейсхолдер) + матеріал + короткий тест → рахує бал і **зберігає прогрес активної дитини** в акаунт. Без входу — урок працює, але пропонує увійти, щоб зберегти прогрес.
- **`docs/uroky.html` — каталог уроків** (новий): згруповано по предметах, бейдж прогресу активної дитини (Пройдено · бал / Не пройдено), кнопки «Почати/Повторити».
- **`docs/kabinet.html` — панель «Навчання дитини»**: показує, скільки уроків пройдено і які (з балами) — **щоб батьки бачили, що дитина займається**. Дані з `lesson_progress` активної дитини.
- Лінки: головна «Більше уроків» і підвал «Уроки» → `uroky.html`.

## Структура одного уроку (каркас)

Заголовок + предмет/клас/тривалість → **відео-пояснення** (YouTube/файл або плейсхолдер) → **матеріал** (HTML-конспект) → **короткий тест на закріплення** (питання з варіантами) → бал → запис у прогрес дитини.

_Тексти й глибину уроків наповнюєте ви з Оксаною: у БД у кожного уроку є поля `video_url`, `material`, і питання в `lesson_questions`._

## Як перевірити

1. Запуш файли (нижче) → жорстке оновлення з очисткою кешу (бо змінювався `app.js`).
2. **Уроки**: у підвалі «Уроки» або на головній «Більше уроків» → каталог із математикою й історією.
3. Відкрий урок → відповідай на тест → «Завершити урок» → побачиш бал і «Збережено в кабінет».
4. **Кабінет** → панель **«Навчання дитини»**: пройдений урок і бал. Прогрес рахується для активної дитини (перемкни дитину — побачиш окремо).
